# Ejecutar en desarrollo

1. Clonar el repositorio

```
git clone https://github.com/Comparitiko/nest-pokedex.git
```

2. Ejecutar

```
pnpm install
```

3. Tener Nest CLI instalado

```
npm i -g @nestjs/cli
```

4. Levantar la base de datos

```
docker-compose up
```

## Stack usado

- MongoDB
- NestJS
- TypeScript
